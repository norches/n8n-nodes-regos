Тиражируемые интеграции

Тиражируемые интеграции добавляются в каталог исключительно сотрудниками REGOS. Свяжитесь с технической поддержкой для уточнения процесса.

Определение и назначение

Тиражируемые интеграции — это универсальные программные модули, созданные для всех пользователей платформы REGOS. Они доступны в каталоге интеграций и предназначены для решения типовых задач, таких как синхронизация данных с внешними сервисами (например, CRM, мессенджерами) или автоматизация процессов (например, отправка уведомлений, обмен документами). Эти интеграции упрощают работу пользователей, предоставляя готовые решения, которые можно установить и настроить под индивидуальные нужды.

Некоторые тиражируемые интеграции являются встраиваемыми, то есть взаимодействуют с внутренними бизнес-процессами REGOS. Такие интеграции поддерживают определённые методы API REGOS (например, SendMessages) и выполняют специализированные функции, такие как отправка SMS или работа с системами электронного документооборота (ЭДО).

С 1 декабря 2025 года каждый запрос, отправляемый тиражируемой интеграцией, должен содержать обязательный заголовок Authorization: Bearer <access_token>. Создайте приложение Едином профиле REGOS; храните их конфиденциально и не передавайте третьим лицам.
