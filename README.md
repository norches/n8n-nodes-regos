# n8n-nodes-regos

n8n community node package for the [REGOS](https://regos.uz) SaaS ERP — retail/ERP platform (inventory, documents, POS, CRM, loyalty) used primarily in Uzbekistan.

**Status: pre-alpha, documentation phase.** No published package yet. The node code will be scaffolded with `npm create @n8n/node` in the next phase; this repository currently holds the project specification, architecture decision records, and vendored REGOS API reference material.

- Project specification: [docs/SPEC.md](docs/SPEC.md)
- Architecture decisions: [docs/adr/](docs/adr/README.md)
- REGOS API reference snapshots: [docs/reference/](docs/reference/README.md)
- API spec of record (codegen input): [openapi/regos_api_swagger.json](openapi/regos_api_swagger.json)
